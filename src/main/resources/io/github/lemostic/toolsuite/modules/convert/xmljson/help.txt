欢迎使用XML/JSON互转工具！

这是一个功能强大的数据格式转换工具，支持XML与JSON之间的双向转换。

功能特点：
1. XML一键转JSON - 将XML数据快速转换为JSON格式
2. JSON一键转XML - 将JSON数据快速转换为XML格式
3. 智能类型检测 - 自动识别输入内容的格式类型
4. 格式化输出 - 支持美化输出，使结果更易阅读
5. 一键复制 - 快速复制转换结果到剪贴板
6. 文件操作 - 支持从文件加载和保存结果到文件
7. 内容交换 - 支持输入输出内容快速交换

使用说明：
1. 在左侧输入框中输入XML或JSON内容
2. 系统会自动检测输入内容的类型
3. 点击中间的转换按钮进行格式转换
4. 转换结果将显示在右侧输出框中
5. 可以使用"复制结果"按钮复制转换结果
6. 可以使用"保存结果"按钮将结果保存为文件

格式支持：
- XML: 标准XML格式，支持嵌套元素和属性
- JSON: 标准JSON格式，支持对象和数组

注意事项：
- 输入的XML/JSON内容必须是有效的格式
- 转换大型数据时可能需要一些时间
- 格式化选项会影响输出结果的排版

快捷键提示：
- 使用"交换"按钮可以快速交换输入输出内容
- 格式化选项可以随时切换，重新转换即可生效
